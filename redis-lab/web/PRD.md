# RedisLab 前端产品需求文档 (100条)

## 对标产品
- **Navicat** — 连接管理、数据浏览、查询编辑器的标杆
- **RedisInsight** — Redis 官方 GUI，Key 浏览器、数据类型查看器
- **DataGrip** — JetBrains 数据库 IDE，查询编辑器、结果集展示
- **TablePlus** — 简洁 UI、快速过滤、行内编辑
- **Slack** — 即时通讯体验标杆
- **ChatGPT** — AI 对话交互标杆

---

## 一、全局布局与导航 (1-15)

1.  **整体布局**: 三栏可拖拽布局（左: 连接+Keys / 中: 查询+结果 / 右: 协作面板），使用 `allotment`，默认比例 260:auto:320，用户拖拽后记住比例到 localStorage
2.  **顶部全局 Header**: 高度 44px（紧凑专业），左侧 Logo+产品名，中部连接选择器，右侧大搜索框+通知+头像
3.  **Header 连接状态指示灯**: 绿色脉冲圆点表示已连接，灰色表示未连接，红色表示连接异常
4.  **Header 全局搜索框**: Cmd+K / Ctrl+K 唤起，支持搜索 Key、命令历史、聊天记录、AI 对话
5.  **左侧面板 Tabs 切换**: Connection / Keys / Monitor 三个 Tab 图标切换，每个 Tab 有独立状态
6.  **左侧面板可折叠**: 点击左面板右边缘或快捷键 Cmd+B 折叠，折叠后显示图标导航条（像 VS Code）
7.  **右侧面板可折叠**: 点击右面板左边缘或快捷键 Cmd+Shift+B 折叠，面板支持 Chat/AI 两个 Tab
8.  **主工作区 Tabs**: 支持多 Tab 页（多个查询窗口），每个 Tab 独立维护编辑器内容和执行结果
9.  **Tab 拖拽排序**: Tab 可拖拽重新排序，右键菜单：关闭/关闭其他/关闭右侧/关闭所有
10. **状态栏**: 底部 22px 状态栏，显示连接信息（host:port）、当前 DB、已用 Key 数、命令执行耗时
11. **响应式断点**: 宽度 < 1024px 时自动折叠右侧面板，< 768px 时左侧面板变为浮层
12. **面包屑导航**: Key 详情模式下，Header 下方显示面包屑 `db0 > user:* > user:1`
13. **快捷键系统**: 统一快捷键管理，所有主要操作有快捷键，`?` 键弹出快捷键面板
14. **通知系统**: 右上角通知 Bell 图标，错误/成功/警告分类显示，支持历史通知列表
15. **右键上下文菜单**: 所有可交互元素（连接、Key、Tab、消息）都有右键菜单

## 二、连接管理 (16-25)

16. **连接列表**: 树形列表，每项显示名称+host:port+状态灯+数据库图标
17. **连接分组**: 支持文件夹/标签分组管理连接（生产/测试/学习）
18. **新建连接 Modal**: 专业表单，名称/主机/端口/密码/DB 选择/SSL 选项/SSH 隧道
19. **连接测试按钮**: 表单底部"测试连接"按钮，3秒超时，显示连接耗时
20. **连接右键菜单**: 编辑/复制/删除/测试连接/刷新/导出连接配置
21. **连接搜索过滤**: 列表顶部搜索框，实时过滤连接名
22. **连接排序**: 支持按名称/最近使用/创建时间排序
23. **最近连接置顶**: 最近使用的3个连接自动置顶，带"最近"标签
24. **连接导入导出**: 支持导出连接配置为 JSON，导入恢复
25. **连接颜色标记**: 可为每个连接设置颜色标签，左侧显示对应色条

## 三、Key 浏览器 (26-40)

26. **Key 树形展示**: 按 `:` 分层，每层节点有类型图标（String/Hash/List/Set/ZSet）
27. **Key 类型图标**: 不同数据类型用不同颜色和图标区分（String=黄色钥匙，Hash=蓝色方块，List=绿色列表，Set=红色集合，ZSet=紫色排序集合）
28. **Key 懒加载**: 展开节点时通过 SCAN 按需加载子节点，避免一次性加载大量 Key
29. **Key 搜索框**: 顶部搜索框，支持通配符 `*` `?`，Enter 搜索，实时过滤
30. **Key 搜索历史**: 搜索框下拉显示最近5次搜索模式
31. **Key 统计信息**: 每个节点显示子 Key 数量 badge
32. **Key 右键菜单**: 查看详情/重命名/删除/设置 TTL/复制 Key 名/复制值/导出
33. **Key 多选**: 支持 Ctrl/Shift 多选 Key，批量删除/批量设置 TTL
34. **Key 刷新按钮**: 顶部刷新图标，点击刷新当前展开的树节点
35. **Key 排序选项**: 按字母/按类型/按大小/按 TTL 排序
36. **Key 信息提示**: Hover 时 Tooltip 显示 Key 全名、类型、TTL、大小
37. **TTL 倒计时显示**: 有 TTL 的 Key 后面显示倒计时（如 "3m25s"），低于 60 秒变橙色，过期变红色
38. **过滤条件**: 支持按类型过滤（只显示 String/只显示 Hash 等）
39. **Key 分页**: 超过 500 个 Key 时分页显示，每页 100 个
40. **空状态设计**: 无 Key 时显示精美空状态插图+引导文字"创建你的第一个 Key"

## 四、查询编辑器 (41-55)

41. **Monaco Editor**: 使用 Monaco Editor，主题 vs-dark，字体 JetBrains Mono 14px
42. **Redis 语法高亮**: 命令名蓝色，Key 名橙色，Value 字符串绿色，数字紫色，注释灰色
43. **智能自动补全**: 输入时弹出命令补全列表，选中的命令显示参数提示（arity, since version）
44. **命令参数提示**: 输入命令名+空格后，显示该命令的完整签名 `SET key value [EX seconds] [PX milliseconds] [NX|XX]`
45. **多行编辑**: 支持 Shift+Enter 换行，一条命令可跨多行
46. **Ctrl+Enter 执行**: 执行当前光标所在行或选中文本
47. **命令历史导航**: 上下箭头在编辑器中浏览历史命令（像终端一样）
48. **历史面板**: 点击工具栏时钟图标，侧边弹出历史命令列表，支持搜索和收藏
49. **收藏命令**: 每条历史命令可点击星标收藏，收藏的命令置顶显示
50. **编辑器工具栏**: 执行按钮(绿)/清除按钮/历史按钮/收藏按钮/放大缩小字体
51. **执行状态**: 执行中按钮显示 loading，编辑器只读，光标变沙漏
52. **语法检查**: 未闭合引号红色波浪线，参数数量错误黄色警告
53. **Pipeline 支持**: 自动识别多行命令中的 Pipeline (多个命令用换行分隔)，一次性发送
54. **选中执行**: 选中部分文本后 Ctrl+Enter 只执行选中部分
55. **字体缩放**: Ctrl+滚轮缩放编辑器字体大小，状态栏显示当前字体大小

## 五、结果展示与数据查看 (56-70)

56. **结果面板**: 编辑器下方，显示执行结果，高度可拖拽调整
57. **类型标签**: 结果顶部显示类型标签（String/Integer/Array/Error/Null），带颜色
58. **String 查看器**: 纯文本渲染，支持自动换行切换，支持 JSON 格式化
59. **Hash 查看器**: 表格展示 field-value，支持排序和搜索
60. **List 查看器**: 序号列表，支持分页（LRANGE 分页加载）
61. **Set 查看器**: 标签云展示 member
62. **ZSet 查看器**: 表格展示 member+score，按 score 排序
63. **结果复制**: 每类结果右上角复制按钮，点击复制完整结果
64. **结果导出**: 支持导出为 JSON/TXT/CSV 格式
65. **执行耗时显示**: 结果底部显示 `⏱ 1.23ms` 执行耗时
66. **错误友好提示**: 错误结果红色背景卡片，显示错误码、错误信息、可能的解决方案
67. **Empty/Null 优雅展示**: (nil) 灰色斜体，(empty list or set) 空状态图标
68. **大结果分页**: 超过 200 条的结果自动分页
69. **结果滚动优化**: 虚拟滚动确保大结果不卡顿
70. **内联编辑**: String 类型值支持双击进入编辑模式，修改后 Ctrl+Enter 保存

## 六、即时通讯 (71-85)

71. **聊天面板布局**: 顶部房间标题栏 + 中间消息列表 + 底部输入框
72. **房间列表**: 左侧面板内嵌房间列表，显示房间名+最后消息预览+未读数 badge
73. **消息气泡**: 对方消息左对齐灰底，自己消息右对齐蓝底，像 iMessage
74. **消息时间戳**: 每条消息显示时间，超过5分钟间隔显示时间分隔线
75. **在线用户列表**: 房间标题栏右侧显示在线人数，点击展开在线用户列表
76. **正在输入提示**: 有人正在输入时显示"xxx 正在输入..."动画
77. **新消息通知**: 不在房间时收到新消息，顶部闪烁通知条
78. **消息滚动**: 新消息自动滚动到底部，查看历史时向上滚动加载更多
79. **消息格式化**: 支持 Markdown 基础语法（代码块、粗体、列表）
80. **代码块渲染**: 消息中的代码块用深色背景+等宽字体渲染，右上角复制按钮
81. **图片/文件分享**: 未来扩展，先留接口
82. **消息搜索**: 房间内搜索历史消息
83. **@提及**: @用户名高亮显示，被 @ 的人收到单独通知
84. **消息持久化滚动位置**: 切换房间后切回，保持上次滚动位置
85. **空聊天引导**: 无消息时显示精美插图+"在 {房间名} 开始交流吧"

## 六、AI 助手 (86-95)

86. **AI 面板布局**: 与聊天面板对称，顶部标题+中间对话+底部输入
87. **对话气泡**: 用户消息蓝底右对齐，AI 消息灰底左对齐，AI 头像为机器人图标
88. **流式打字效果**: AI 回复逐字显示，光标闪烁动画
89. **Markdown 渲染**: AI 回复完整支持 Markdown（标题/列表/链接/代码/表格）
90. **代码块增强**: 代码块显示语言标签+一键复制按钮+运行按钮（可执行 Redis 命令）
91. **上下文感知**: AI 自动感知当前连接的 Redis 信息（版本/Key数量/最近命令），无需用户手动提供
92. **会话管理**: 左侧会话列表，支持新建/重命名/删除会话
93. **会话搜索**: 搜索历史对话内容
94. **建议问题**: AI 面板为空时显示 3-5 个建议问题卡片
95. **停止生成按钮**: AI 回复过程中显示停止按钮，点击中断流式输出

## 七、主题、动效与打磨 (96-100)

96. **暗色主题优先**: 默认暗色主题 #0a0b10 底色，所有组件深色适配，对比度符合 WCAG AA
97. **亮色主题**: 提供亮色主题切换，一键切换暗/亮
98. **过渡动画**: Tab 切换/面板展开折叠/消息出现有 200ms ease 过渡
99. **骨架屏**: 数据加载时显示骨架屏占位而不是空白或转圈
100. **微交互**: 按钮 hover 缩放 1.02、点击缩放 0.98、图标 hover 颜色变化、Toast 弹出从右滑入
